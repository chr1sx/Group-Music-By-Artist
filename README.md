#     Group Music By Artist

[![Download Script](https://img.shields.io/badge/Download%20Script-blue?style=for-the-badge)]([https://github.com/chr1sx/Audio-To-Discogs-CSV-Exporter/archive/refs/heads/main.zip](https://github.com/chr1sx/Group-Music-By-Artist/archive/refs/heads/main.zip))

*Automatically organize music folders by artist.*

</div>

---

## Features

- Handles collaborations: `&`, `feat.`, `vs.`, `x`, `,`, `+`, etc. all group to the primary artist
- Works at any depth in subdirectories
- Two organization modes: organize all albums, or only artists with multiple albums
- Preview before moving

---

## Usage

1. Install Python 3
2. Drag and drop a music folder onto the script
3. Choose organization option and confirm

**Example:**
```
Before:
├── Artist A - Album 1/
├── Artist A - Album 2/
└── Artist A feat. Guest - Collab/

After:
└── Artist A/
    ├── Artist A - Album 1/
    ├── Artist A - Album 2/
    └── Artist A feat. Guest - Collab/
```
